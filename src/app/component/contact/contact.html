Explore Conditional Rendering and Two-Way Binding in Angular

1️⃣ Age Check with @if

@if (myinfo.age < 30) {

✔️ Age is less than 30

} @else {

⚠️ Age is 30 or more

}

2️⃣ Age Categories with @switch

@switch (myinfo.age) { @case (20) {

👶 Age is exactly 20

} @case (30) {

🧑 Age is exactly 30

} @default {

🧓 Age is more than 30

} }
